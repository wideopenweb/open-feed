# open-feed
Open Feed aims to provide push/pull synchronization with major social media platforms and an open, portable format.

## Roadmap
1. Pull from Facebook
2. Pull from ...
3. Push to Facebook
4. Push to ...
5. Publish to open-feed
6. View on open-feed
7. Distributed hosting
8. iOS App
9. Android App
